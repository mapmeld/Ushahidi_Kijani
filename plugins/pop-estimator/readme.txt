name: Population Estimator
website: http://www.ushahidi.com
description: Draw a polygon, get population estimate from ArcGIS Server API
version: 0.6
requires: 2.0
tested up to: 2.0
author: Nick Doiron
author website: http://andrew.cmu.edu/~ndoiron/

== Description ==
* Click the tool in the right sidebar of Ushahidi to activate
* Click on the map to start drawing a polygon shape
* Double-click to stop drawing
* In a few moments, the ArcGIS Server API will return a population estimate
